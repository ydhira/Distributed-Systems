******************************
********* How to Run *********
******************************

* kMean2D.java
- MapReduce of KMeans on 2D points 
Run :  time hadoop jar kMean2D.jar input output t n
	input: the input directory
	output: the outfut directory
	t: total number of points
	n : the number of clusters 

* kMeandna.java
- MapReduce of KMeans on DNA points 
Run : time hadoop jar kMeandna.jar kMeandna input output n
	input: the input directory
	output: the outfut directory
	n : the number of clusters
